0. Make sure your laptop's system sound is set to use the MiniFuse
1. Open a command prompt: Terminal on Mac, Windows Terminal on Windows
2. Change to the directory that holds the Clix files: cd ~/Documents/GitHub/MULE-Ox/F25-Kit/clix
3. Run the following command: ls
4. You should see a list of files with "clix" in the name
5. Run the following command: chuck clix.ck
6. If it worked, you should momentarily see the message "keyboard ready..."
