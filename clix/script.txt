0. Make sure your laptop's system sound is set to use the MiniFuse
1. Right-click the clix folder and, if available, choose "New Terminal at Folder" (Mac) or "Open in Terminal" (Windows).
2. If you don't see that option on Windows, copy the folder, open Terminal from the search menu (or Command Prompt or Powershell), type cd, then space, then paste. Then hit return.
3. Run the following command: ls
4. You should see a list of files with "clix" in the name
5. Run the following command: chuck clix.ck
6. If it worked, you should momentarily see the message "keyboard ready..."
